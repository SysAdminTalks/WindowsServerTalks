Authentication Flow – User or Computer Authentication

When “User or Computer authentication” is configured, Windows follows a conditional, state‑based authentication sequence rather than running both identities simultaneously.

Order of Evaluation (What is checked first)

Computer Authentication (Checked first – during system startup)
When the machine boots and no user is logged in, the system attempts machine (computer account) authentication.

Uses:

  • Computer certificate (EAP-TLS) OR
  • Machine credentials (PEAP-MSCHAPv2 depending on configuration)
Purpose:
  • Enables network access before login
  • Allows GPOs, scripts, and domain communication
User Authentication (Triggered after user logon)
Once a user logs in, the system switches to user authentication.

Uses:
  • User credentials or user certificate
Purpose:
  • Applies user-based policies
  • Enables user-level network access tracking
Fallback Behaviour
  • If user authentication fails, the system may fall back to computer authentication (depending on EAP settings and server policy)
  • If the user logs off, the system reverts back to computer authentication

How to test yourself what GUI changes does in XML ?

Use the below script to check the changes happening in xml profile

How the Script works ?

When you execute the script for the first time it caches the current XML and then you need to make the profile changes in GUI and rerun the script. The script then export the current profile XML and checks the portion removed and added and give you the details in a notepad file, which opens as a popup.

Script


<#
run this first Set-ExecutionPolicy -scope Process -ExecutionPolicy RemoteSigned -Force
.SYNOPSIS
    Export WLAN profile "test" to C:\Wifi\test.xml, rotate previous to test.old.xml,
    compute clean structural diffs (namespace-agnostic; ignores xmlns),
    write results to difference.xml and difference.txt,
    and open difference.txt in Notepad (no popups).

.NOTES
    Run PowerShell as Administrator if writing to C:\Wifi requires elevation.
#>

[CmdletBinding()]
param(
    [string]$ProfileName = 'test',
    [string]$Folder      = 'C:\Wifi'
)

$ErrorActionPreference = 'Stop'

# Canonical file paths
$NewFile   = Join-Path $Folder 'test.xml'
$OldFile   = Join-Path $Folder 'test.old.xml'
$DiffXml   = Join-Path $Folder 'difference.xml'
$DiffTxt   = Join-Path $Folder 'difference.txt'

# Ensure destination folder
if (-not (Test-Path -LiteralPath $Folder)) {
    New-Item -Path $Folder -ItemType Directory -Force | Out-Null
}

# Step 2: delete test.old.xml if present
if (Test-Path -LiteralPath $OldFile) {
    Remove-Item -LiteralPath $OldFile -Force
}

# Step 3: if test.xml exists, rename to test.old.xml
$Renamed = $false
if (Test-Path -LiteralPath $NewFile) {
    Rename-Item -LiteralPath $NewFile -NewName (Split-Path -Path $OldFile -Leaf) -Force
    $Renamed = $true
}

# Step 4: Export WLAN profile "test" to C:\Wifi\test.xml
$exportStart = Get-Date
& netsh wlan export profile name="$ProfileName" key=clear folder="$Folder" | Out-Null

$exported = Get-ChildItem -Path $Folder -Filter '*.xml' -ErrorAction SilentlyContinue |
    Where-Object { $_.LastWriteTime -ge $exportStart.AddSeconds(-1) } |
    Sort-Object LastWriteTime -Descending |
    Select-Object -First 1

if (-not $exported) {
    throw "Export failed: netsh did not create any XML file in '$Folder'. Ensure the WLAN profile '$ProfileName' exists."
}

Move-Item -LiteralPath $exported.FullName -Destination $NewFile -Force

function Load-XmlDoc {
    param([Parameter(Mandatory)][string]$Path)
    $settings = New-Object System.Xml.XmlReaderSettings
    $settings.IgnoreWhitespace = $true
    $settings.IgnoreComments   = $true
    $settings.IgnoreProcessingInstructions = $true
    $reader = [System.Xml.XmlReader]::Create($Path, $settings)
    $doc = New-Object System.Xml.XmlDocument
    $doc.PreserveWhitespace = $false
    $doc.Load($reader)
    $reader.Close()
    return $doc
}

function Get-LocalPath {
    param([System.Xml.XmlNode]$node)

    if (-not $node) { return '/' }

    $parts = @()
    $cur = $node
    while ($cur -and $cur.NodeType -ne [System.Xml.XmlNodeType]::Document) {
        if ($cur.NodeType -eq [System.Xml.XmlNodeType]::Element) {
            # Compute sibling index by iterating elements with same LocalName
            $idx = 1
            $parent = $cur.ParentNode
            if ($parent -and $parent.HasChildNodes) {
                $idx = 0
                foreach ($s in $parent.ChildNodes) {
                    if ($s.NodeType -eq 'Element' -and $s.LocalName -eq $cur.LocalName) {
                        $idx++
                        if ([Object]::ReferenceEquals($s, $cur)) { break }
                    }
                }
                if ($idx -le 0) { $idx = 1 }
            }
            $parts += ('{0}[{1}]' -f $cur.LocalName, $idx)
        }
        elseif ($cur.NodeType -eq [System.Xml.XmlNodeType]::Attribute) {
            $parts += ('@{0}' -f $cur.LocalName)
        }
        $cur = $cur.ParentNode
    }

    if ($parts.Count -eq 0) { return '/' }
    return '/' + ($parts[-1..-($parts.Count)] -join '/')
}

function Build-LeafMap {
    param([System.Xml.XmlDocument]$doc)

    $map = @{}

    function Add-AttributeEntries {
        param([System.Xml.XmlElement]$elem)
        if ($elem.Attributes -and $elem.Attributes.Count -gt 0) {
            foreach ($a in $elem.Attributes) {
                # Ignore xmlns noise
                if ($a.Name -eq 'xmlns' -or $a.Prefix -eq 'xmlns') { continue }
                $pathElem = Get-LocalPath -node $elem
                $pathAttr = "$pathElem/@$($a.LocalName)"
                $val      = [string]$a.Value
                $snippet  = '@{0}="{1}"' -f $a.LocalName, $val
                $map[$pathAttr] = [pscustomobject]@{
                    Name        = "@$($a.LocalName)"
                    Value       = $val
                    Snippet     = $snippet
                    IsAttribute = $true
                }
            }
        }
    }

    function Recurse {
        param([System.Xml.XmlNode]$n)

        if ($n.NodeType -eq [System.Xml.XmlNodeType]::Element) {
            $elem = [System.Xml.XmlElement]$n

            Add-AttributeEntries -elem $elem

            # Leaf = no element children
            $hasElemChildren = $false
            foreach ($ch in $elem.ChildNodes) {
                if ($ch.NodeType -eq 'Element') { $hasElemChildren = $true; break }
            }

            if (-not $hasElemChildren) {
                $val = [string]$elem.InnerText
                if ($null -ne $val) { $val = $val.Trim() } else { $val = '' }
                $path = Get-LocalPath -node $elem
                $name = $elem.LocalName
                $snippet = "<{0}>{1}" -f $name, $val

                $map[$path] = [pscustomobject]@{
                    Name        = $name
                    Value       = $val
                    Snippet     = $snippet
                    IsAttribute = $false
                }
            }

            foreach ($ch in $elem.ChildNodes) { Recurse -n $ch }
        }
    }

    Recurse -n $doc.DocumentElement
    return $map
}

function Write-Differences {
    param(
        [hashtable]$OldMap,
        [hashtable]$NewMap,
        [string]$XmlPath,
        [string]$TxtPath
    )

    $generatedAt = (Get-Date).ToString('yyyy-MM-dd HH:mm:ss.fff')

    $xml = New-Object System.Text.StringBuilder
    [void]$xml.AppendLine('')
    [void]$xml.AppendLine("")

    $txt = New-Object System.Collections.Generic.List[string]

    $allKeys = @($OldMap.Keys + $NewMap.Keys) | Select-Object -Unique | Sort-Object
    $changeCount = 0

    foreach ($k in $allKeys) {
        $oldItem = $OldMap[$k]
        $newItem = $NewMap[$k]

        if ($null -eq $oldItem -and $null -ne $newItem) {
            # ADDED
            $changeCount++
            $ts = (Get-Date).ToString('HH:mm:ss.fff')
            $newSafe = $newItem.Snippet -replace ']]>', ']]]]>'

            [void]$xml.AppendLine("  ")
            [void]$xml.AppendLine("    $k")
            [void]$xml.AppendLine("    $($newItem.Name)")
            [void]$xml.AppendLine("    ")
            [void]$xml.AppendLine("    ")
            [void]$xml.AppendLine("  ")

            $txt.Add(('new change "{0}"' -f $newItem.Snippet))
        }
        elseif ($null -ne $oldItem -and $null -eq $newItem) {
            # REMOVED
            $changeCount++
            $ts = (Get-Date).ToString('HH:mm:ss.fff')
            $oldSafe = $oldItem.Snippet -replace ']]>', ']]]]>'

            [void]$xml.AppendLine("  ")
            [void]$xml.AppendLine("    $k")
            [void]$xml.AppendLine("    $($oldItem.Name)")
            [void]$xml.AppendLine("    ")
            [void]$xml.AppendLine("    ")
            [void]$xml.AppendLine("  ")

            $txt.Add(('removed "{0}"' -f $oldItem.Snippet))
        }
        elseif ($null -ne $oldItem -and $null -ne $newItem) {
            # CHANGED
            if ($oldItem.Value -ne $newItem.Value) {
                $changeCount++
                $ts = (Get-Date).ToString('HH:mm:ss.fff')
                $oldSafe = $oldItem.Snippet -replace ']]>', ']]]]>'
                $newSafe = $newItem.Snippet -replace ']]>', ']]]]>'

                [void]$xml.AppendLine("  ")
                [void]$xml.AppendLine("    $k")
                [void]$xml.AppendLine("    $($newItem.Name)")
                [void]$xml.AppendLine("    ")
                [void]$xml.AppendLine("    ")
                [void]$xml.AppendLine("  ")

                $txt.Add(('new change "{0}"' -f $newItem.Snippet))
            }
        }
    }

    if ($changeCount -eq 0) {
        $ts = (Get-Date).ToString('HH:mm:ss.fff')
        [void]$xml.AppendLine("  /No differences found")
        $txt.Add("No differences found")
    }

    [void]$xml.AppendLine("")

    Set-Content -LiteralPath $XmlPath -Value $xml.ToString() -Encoding UTF8
    Set-Content -LiteralPath $TxtPath -Value $txt -Encoding UTF8

    if (-not (Test-Path -LiteralPath $XmlPath)) { Write-Warning "Could not find $XmlPath after writing." }
    if (-not (Test-Path -LiteralPath $TxtPath)) { Write-Warning "Could not find $TxtPath after writing." }
}

function Do-Diff {
    param(
        [string]$OldPath,
        [string]$NewPath,
        [string]$XmlPath,
        [string]$TxtPath
    )

    $oldDoc = Load-XmlDoc -Path $OldPath
    $newDoc = Load-XmlDoc -Path $NewPath

    $oldMap = Build-LeafMap -doc $oldDoc
    $newMap = Build-LeafMap -doc $newDoc

    Write-Differences -OldMap $oldMap -NewMap $newMap -XmlPath $XmlPath -TxtPath $TxtPath

    # Open difference.txt robustly (no popup)
    if (Test-Path -LiteralPath $TxtPath) {
        try {
            Invoke-Item -LiteralPath $TxtPath
        } catch {
            try {
                Start-Process -FilePath "notepad.exe" -ArgumentList @("$TxtPath") -WindowStyle Normal | Out-Null
            } catch {
                Write-Warning "Failed to open '$TxtPath'. Please open it manually."
            }
        }
    } else {
        Write-Warning "Expected '$TxtPath' was not created. Please re-run the script."
    }
}

# Run diff if we have an old copy; otherwise write stub and open it
if ($Renamed -and (Test-Path -LiteralPath $OldFile)) {
    Do-Diff -OldPath $OldFile -NewPath $NewFile -XmlPath $DiffXml -TxtPath $DiffTxt

    }
}
